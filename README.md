# 2048 Neon Edition 🎮

A futuristic neon-themed 2048 game built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Neon cyberpunk UI
- Smooth animations
- Responsive design
- Mobile swipe support
- Keyboard controls
- Score tracking
- Best score saving using LocalStorage
- Undo move feature
- Particle effects
- Game over overlay
- Modern landing page design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎮 Controls

### Keyboard

| Key | Action |
|-----|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
| W | Move Up |
| S | Move Down |
| A | Move Left |
| D | Move Right |

### Mobile

- Swipe in any direction to move tiles

## 🎯 Game Rules

- Combine tiles with the same number
- Each merge doubles the tile value
- Reach the **2048** tile to win
- Game ends when no moves remain
