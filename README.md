# 🎮 2048 Game

A fun and addictive browser-based implementation of the classic 2048 puzzle game.

## 🚀 Features
- 🎯 Smooth tile animations
- 📱 Mobile touch support
- 🏆 High score tracking
- ⌨️ Keyboard controls
- 🔄 Restart anytime
